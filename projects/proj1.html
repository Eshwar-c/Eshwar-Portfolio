Personal Portfolio Website | HTML, CSS & JavaScript

A modern and responsive portfolio website crafted from scratch to showcase my web development projects. Built using semantic HTML5 for optimal structure, CSS3 with Flexbox and Grid layouts for seamless responsiveness, and vanilla JavaScript for interactive elements. The clean, minimalist design ensures visitors can easily navigate through my work samples while demonstrating my front-end development capabilities. Special attention was given to user experience and cross-browser compatibility.

    Key Features
  1. Responsive design that adapts seamlessly across all devices
  2. Project showcase section with detailed project cards
  3. Clean and intuitive navigation using HTML5 semantic elements
  4. Modern layout implementation using CSS Flexbox and Grid
  5. Interactive elements powered by vanilla JavaScript
